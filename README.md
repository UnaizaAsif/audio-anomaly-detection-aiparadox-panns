# Audio Anomaly Detection - AiParadox

This repository contains the AiParadox project for audio anomaly detection using PANNs (Pretrained Audio Neural Networks) combined with handcrafted audio features, SMOTE balancing, and ensemble gradient boosting models.

## Project Structure

- `report/` : Final project report (PDF)
- `panns_advanced.ipynb` : Jupyter notebook for experiments and analysis
- `panns_advanced.py` : Python script for feature extraction, model training, and inference
- `submission/` : CSV files ready for Kaggle submission
